# `lis331-rs`

[![Crate][crate-image]][crate-link]
[![Docs][docs-image]][docs-link]
![MIT licensed][license-image]

Platform-agnostic Rust driver for the ST LIS331 (DLH, HH) 3-axis MEMS "nano" accelerometer.
